# PatternWise Legal Documents

This directory contains the legal documents for PatternWise app, hosted on GitHub Pages.

## Files

- `privacy.md` / `privacy.html` - Privacy Policy
- `terms.md` / `terms.html` - Terms of Use
- `index.html` - Landing page with links to legal documents

## GitHub Pages Setup

1. Push this `docs/` folder to your GitHub repository
2. Go to repository Settings > Pages
3. Select "Deploy from a branch"
4. Select branch: `main` (or your default branch)
5. Select folder: `/docs`
6. Click Save

Your legal documents will be available at:
- `https://[username].github.io/[repo-name]/privacy.html`
- `https://[username].github.io/[repo-name]/terms.html`

## Usage in App

Update your app's configuration to use these URLs:
- Privacy Policy: `https://[username].github.io/[repo-name]/privacy.html`
- Terms of Use: `https://[username].github.io/[repo-name]/terms.html`

## App Store Connect

Use these URLs in App Store Connect:
- Privacy Policy URL: `https://[username].github.io/[repo-name]/privacy.html`
- Terms of Use URL: `https://[username].github.io/[repo-name]/terms.html`
